SHRTME Microservice

    Project Description
  • I can pass a URL as a parameter and I will receive a generated URL in the JSON response.
  • When I visit that generated URL, it will redirect me to my original link.

Example creation usage:

https://shrtme.glitch.me/new/https://www.google.com
https://shrtme.glitch.me/new/https://www.freeCodeCamp.org

Example creation output

{ "originalUrl":"https://www.google.com", "short_url":"http://shrtme.glitch.me/9NK&0CRoU" }

Usage:

http://shrtme.glitch.me/9NK&0CRoU

Will redirect to:

https://www.google.com/
